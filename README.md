## 📌 Sobre o projeto

Aplicação web construída com **Python e Flask** que cobre o fluxo de autenticação de usuários: cadastro, login, dashboard e alteração de senha. O front-end é organizado em templates HTML com arquivos estáticos de CSS e JavaScript.

## ✨ Funcionalidades

- 🧾 Cadastro de usuários
- 🔑 Login
- 📊 Dashboard pós-login
- 🔄 Alteração de senha

## 🛠️ Tecnologias

Python · Flask · HTML · CSS · JavaScript · SQL

## 🗂️ Estrutura

```
app.py             # aplicação Flask (rotas)
database.sql       # script do banco de dados
templates/         # páginas (index, login, cadastro, dashboard, alterar_senha)
static/css|js/     # estilos e scripts
```

## 🚀 Como executar

```bash
git clone https://github.com/Samuelf27/meu_sistema.git
cd meu_sistema

pip install flask

python app.py
```

Acesse: **http://localhost:5000**
